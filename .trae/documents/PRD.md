# 局域网应急协同桌面客户端 - 产品需求文档

## 1. 产品概述

一款专为工厂、医院、展会等网络受限环境设计的局域网应急协同桌面客户端，支持事件频道管理、即时通讯、文件共享、成员状态监控和任务看板等功能，实现快速沟通与资料交换。

### 产品目标
- 在无互联网环境下提供稳定高效的团队协作能力
- 支持多岗位人员快速加入事件响应频道
- 实现文件、图片、语音消息的即时共享与归档
- 确保紧急情况下的通信可靠性和消息传递确认

### 目标用户
- 工厂车间管理人员和技术人员
- 医院急诊和手术室人员
- 展会现场工作人员和志愿者
- 企业应急响应团队

## 2. 核心功能模块

### 2.1 用户角色
| 角色 | 权限范围 | 说明 |
|------|---------|------|
| 管理员 | 全部功能、用户管理、频道管理 | 创建和管理事件频道，管理系统用户 |
| 主管 | 本频道管理、文件分发、待办指派 | 管理和监控特定事件频道 |
| 成员 | 加入频道、发送消息、上传文件 | 普通参与人员，可加入多个频道 |

### 2.2 五大核心界面

#### 2.2.1 频道列表界面
- 左侧固定导航栏，显示所有可用频道
- 频道分类：按事件类型（紧急事件、任务协作、一般讨论）
- 频道状态指示：活跃度、未读消息数、成员在线数
- 快速搜索频道功能
- 创建新频道入口

#### 2.2.2 消息面板界面
- 即时消息输入与显示区域
- 支持文字、语音、图片、文件消息
- @提及功能，支持选择提醒人员
- 消息已读/未读状态标记
- 历史消息时间线导航
- 消息搜索功能

#### 2.2.3 文件箱界面
- 按事件和日期归档的文件列表
- 文件上传：支持拖拽上传、批量上传
- 文件类型筛选：图片、文档、压缩包
- 文件分发确认状态查看
- 过期文件清理提醒
- 离线缓存设置

#### 2.2.4 人员状态界面
- 成员列表与在线状态显示
- 岗位/部门信息展示
- 成员在线/离线/忙碌状态
- 按状态筛选成员
- 成员详细信息弹窗
- 紧急联系人快速拨号

#### 2.2.5 事件看板界面
- 看板式任务管理视图
- 待办事项创建与指派
- 任务状态流转：待处理→进行中→已完成
- 任务优先级标记
- 任务截止日期提醒
- 任务完成确认功能

### 2.3 核心业务流程

#### 2.3.1 事件响应流程
```
发现事件 → 创建事件频道 → 邀请相关人员 → 发布事件信息 → 分配待办任务 → 实时沟通协作 → 归档文件记录 → 关闭事件频道
```

#### 2.3.2 文件共享流程
```
上传文件 → 选择分发范围 → 确认发送 → 接收方收到提醒 → 查看/下载文件 → 确认接收 → 文件自动归档
```

## 3. 功能详细说明

### 3.1 频道管理
- **创建频道**：输入频道名称、描述、分类、初始成员
- **加入频道**：按岗位自动匹配或手动搜索加入
- **频道设置**：设置管理员、修改信息、设置权限
- **频道归档**：事件结束后归档频道，保留记录

### 3.2 消息通讯
- **文字消息**：支持富文本，包含链接、代码片段
- **语音留言**：录音上传，时长限制3分钟
- **图片消息**：支持拍照上传，自动压缩
- **文件消息**：支持常见办公文档格式
- **已读回执**：显示消息送达和已读状态
- **@提及**：输入@触发人员选择器

### 3.3 文件管理
- **上传功能**：拖拽上传、批量上传、进度显示
- **文件分发**：选择接收成员，发送确认请求
- **接收确认**：查看谁已接收/未接收文件
- **按事件归档**：自动按频道和时间归档
- **离线缓存**：设置缓存大小限制和自动清理策略
- **过期清理**：设置文件保留期限，自动清理过期文件

### 3.4 成员管理
- **状态显示**：在线（绿色）、离线（灰色）、忙碌（红色）、离开（黄色）
- **状态设置**：手动设置当前状态
- **在线提醒**：成员上线/下线通知
- **人员搜索**：按姓名、岗位、状态筛选

### 3.5 任务看板
- **创建待办**：标题、描述、负责人、截止日期、优先级
- **优先级**：紧急（红色）、重要（橙色）、一般（蓝色）
- **状态流转**：拖拽改变任务状态
- **任务提醒**：到期前提醒负责人
- **完成确认**：负责人确认任务完成

### 3.6 数据导出
- **聊天记录导出**：按时间范围和频道导出
- **文件列表导出**：Excel/CSV格式
- **归档文件下载**：打包下载事件相关文件

## 4. 用户界面设计

### 4.1 设计风格
**风格定位**：工业实用主义 + 应急响应主题

**设计理念**：
- 高对比度、高可读性，适合各种光线条件
- 简洁直接，减少操作步骤
- 突出紧急状态和重要信息
- 适应专业工作环境

### 4.2 配色方案
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色（深蓝） | 专业、稳重 | #1E3A5F |
| 次色（灰蓝） | 背景、辅助 | #2C3E50 |
| 强调色（橙色） | 警告、紧急 | #E67E22 |
| 成功色（绿色） | 在线、成功 | #27AE60 |
| 危险色（红色） | 紧急、错误 | #E74C3C |
| 文字（白色） | 主要文字 | #FFFFFF |
| 文字（浅灰） | 次要文字 | #BDC3C7 |

### 4.3 字体选择
- **标题字体**：思源黑体 Bold / Noto Sans SC Bold
- **正文字体**：思源黑体 Regular / Noto Sans SC Regular
- **等宽字体**：Roboto Mono（用于代码、时间戳）
- **字号体系**：标题 24px、正文 14px、辅助 12px

### 4.4 布局结构
```
┌─────────────────────────────────────────────────────────┐
│  顶部工具栏（Logo、搜索、全局操作、用户信息）              │
├────────┬────────────────────────────────────────────────┤
│        │                                                │
│  左侧  │              主内容区域                         │
│  导航  │         （根据选中模块显示）                    │
│  栏    │                                                │
│        │                                                │
│  频道  │  消息面板 / 文件箱 / 人员状态 / 事件看板        │
│  列表  │                                                │
│        │                                                │
├────────┴────────────────────────────────────────────────┤
│  底部状态栏（连接状态、同步状态、版本信息）                │
└─────────────────────────────────────────────────────────┘
```

### 4.5 响应式策略
- **桌面优先**：主要针对 1280px+ 屏幕优化
- **平板适配**：1024px 以下采用紧凑布局
- **功能优先**：移动端仅支持查看和基础操作

### 4.6 交互动效
- **页面切换**：平滑过渡，200ms ease-out
- **悬停效果**：微妙阴影提升，150ms
- **消息到达**：左侧滑入 + 微弹效果
- **状态变化**：颜色渐变过渡
- **加载状态**：骨架屏 + 脉冲动画

## 5. 性能与安全要求

### 5.1 性能指标
- 首屏加载时间 < 3秒
- 消息发送延迟 < 500ms
- 文件列表加载 < 1秒
- 离线缓存支持 500MB+

### 5.2 安全考虑
- 局域网直连，无需公网传输
- 本地数据加密存储
- 操作日志记录
- 敏感操作二次确认

## 6. 技术约束

- 纯前端实现，无需后端服务器
- 使用本地存储模拟数据持久化
- Web Audio API 实现语音录制
- IndexedDB 存储离线缓存数据
- 模拟局域网通信机制
